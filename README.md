# 자연어 처리를 위한 넓고 얕은 튜토리얼

자연어 처리와 관련된 넓고 얕은 튜토리얼 세션들을 관리하는 저장소입니다.

한국어로 작성된 좋은 튜토리얼 자료들이 많이 축적되어 지식이 보다 쉽게 공유되기를 희망합니다.
